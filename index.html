HTML

  • HTML is like the foundation of a house, provides the "base" for the website
  • The Header element contains info about the page (hidden from end user)
  • The Body element contains all the stuff the end user sees!
  • Any elements (likes < ul > for unordered lists) that are contained in an element ( like < body >) are children to the parent element!
