HTML

  • HTML is like the foundation of a house, provides the "base" for the website.
  • The Header element contains info about the page (hidden from end user).
  • The Body element contains all the stuff the end user sees!
  • Any elements (likes < ul > for unordered lists) that are contained in an element ( like < body >) are children to the parent element!

CSS

  • CSS is like the paint for your house, giving it style.
  • A margin indicates how much space we want around an element.
  • A padding indicates how much space we around the content inside an element.
  • The most popular way to use CSS to style a page is an external CSS stylesheet. It makes it easier to do design changes!
  • The general CSS rule layout is selector (which defines which attribute) followed by your declarations(rules your attribute will follow).

GIT

  • GIT is a version control system.
  • the command, git status: checks what branch we are currently working on.
  • the command, git checkout -b "branch name": creates a new branch and switches to it.

JavaScript

  • JavaScript is like the plumbing, electricity and door of your house, making it interactive.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • One of the main "building blocks" of JavaScript are functions.
  • Functions allow us to reuse blocks of code. Not only must you define what your functions does, you need to call it before it will execute!
